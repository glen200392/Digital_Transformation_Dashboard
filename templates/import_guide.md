# Excel/CSV 檔案導入指南

## 📋 概述

本指南說明如何使用 Excel 或 CSV 檔案批次導入資料到數位轉型儀表板。

## 🎯 支援的資料類型

1. **專案 (Projects)**
2. **風險 (Risks)**
3. **Quick Wins**
4. **KPI 指標**

## 📥 下載範本

在資料管理面板中，點擊對應的「下載範本」按鈕，即可取得標準化的 Excel 範本。

## 📤 上傳步驟

1. **下載範本**
   - 點擊對應的「下載範本」按鈕
   - 取得 Excel 檔案

2. **填寫資料**
   - 在範本中填入您的資料
   - 確保必填欄位都有填寫
   - 日期格式建議: YYYY-MM-DD

3. **儲存檔案**
   - 儲存為 .xlsx, .xls 或 .csv 格式
   - 檔案大小限制: 5MB
   - 資料筆數限制: 1000 筆

4. **上傳檔案**
   - 選擇資料類型
   - 拖放檔案或點擊選擇檔案
   - 等待系統解析

5. **檢視預覽**
   - 查看資料預覽表格
   - 檢查驗證結果
   - 確認錯誤和警告訊息

6. **確認導入**
   - 確認無誤後點擊「確認導入」
   - 等待導入完成
   - 查看成功訊息

## ⚠️ 注意事項

### 資料格式要求

- **日期格式**: YYYY-MM-DD (例如: 2024-12-31)
- **數字格式**: 不要使用千分位符號
- **狀態值**: 使用小寫英文 (如 in_progress 而非 In Progress)
- **進度值**: 0-100 之間的數字
- **風險等級**: low/medium/high 或 1-5

### 常見錯誤

1. **必填欄位遺漏**: 確保所有標記為必填的欄位都有填寫
2. **日期格式錯誤**: 使用 YYYY-MM-DD 格式
3. **數值範圍錯誤**: 進度必須在 0-100 之間
4. **狀態值不正確**: 使用預定義的狀態值
5. **檔案過大**: 超過 5MB 限制

### 驗證結果說明

- ✅ **綠色勾號**: 資料通過驗證
- ⚠️ **黃色警告**: 有警告訊息但可以導入
- ❌ **紅色叉號**: 有錯誤，需要修正

### 導入模式

- **追加模式** (預設): 新資料會加到現有資料後面
- 不會覆蓋或刪除現有資料

## 💡 最佳實踐

1. **使用範本**: 始終從下載的範本開始
2. **小批次導入**: 建議每次不超過 100 筆
3. **先測試**: 先用少量資料測試
4. **檢查預覽**: 導入前仔細檢查預覽表格
5. **備份資料**: 重要資料請先備份

## 🔧 疑難排解

### 檔案無法上傳
- 檢查檔案格式是否正確 (.xlsx, .xls, .csv)
- 確認檔案大小未超過 5MB
- 嘗試重新下載範本並複製資料

### 驗證失敗
- 查看錯誤訊息詳情
- 檢查必填欄位是否完整
- 確認資料格式符合要求

### 導入失敗
- 檢查網路連線
- 確認有導入權限
- 聯繫系統管理員
